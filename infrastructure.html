Schéma d'Infrastructure Réseau

Architecture réseau

Voici le schéma d'infrastructure réseau que j'ai conçu et déployé dans le cadre de mes projets BTS SIO SISR.

Pour afficher votre schéma réseau :

1. Placez votre fichier JPG dans le même dossier que vos fichiers HTML

2. Nommez-le "schema-reseau.jpg"

3. Rafraîchissez cette page

Format accepté : JPG/JPEG
Résolution recommandée : 1920x1080 ou supérieure

Description de l'infrastructure

Cette infrastructure comprend :

  • Contrôleur de domaine Active Directory avec DNS intégré
  • Serveur DHCP pour l'attribution automatique des adresses IP
  • Serveur de fichiers (SRV-FIC) pour la centralisation des données
  • Postes clients Windows 10/11 joints au domaine
  • Architecture réseau sécurisée avec segmentation appropriée

Technologies utilisées

Windows Server 2019 Active Directory DNS DHCP VMware VirtualBox Serveur de fichiers
